Check shipping availability and delivery options for your WooCommerce store using pincode/zipcode verification system.

== Description ==

 Delivery Postcode Validator For Shipping allows customers to verify shipping availability to their location by entering their pincode/zipcode. This WordPress plugin provides detailed delivery information including estimated delivery days and Cash on Delivery (COD) availability for each location.

= Key Features =

* **Shipping Availability Check**: Instantly verify if delivery is available for a given pincode.
* **Delivery Time Estimation**: Display estimated delivery days for each pincode.
* **COD Management**: 
    * Enable/Disable COD for specific pincodes.
    * Set different COD charges for each pincode.
    * Display COD charges on cart and checkout pages.
* **Flexible Implementation**:
    * Easy pincode data import via CSV.
    * Manual pincode data entry option.
    * Customizable pincode checker box placement using shortcode.
* **Customization Options**:
    * Modify button and label text.
    * Enable/Disable delivery date display.
    * Flexible positioning of the pincode checker box using WordPress shortcodes.

= Usage =

1. Install and activate the plugin.
2. Import your pincode data via CSV or add manually.
3. Configure display settings and customization options.
4. Place the pincode checker on product pages using shortcode (optional).

== Installation ==

1. Upload 'wc-pincode-checker' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under WooCommerce > Pincode Checker.

== Frequently Asked Questions ==

= Is CSV import available for pincode data? =
Yes, you can import pincode data using CSV files for bulk upload.

= Can I customize the position of the pincode checker? =
Yes, you can either use the built-in position settings or place it anywhere using the shortcode provided for WordPress integration.

= Is COD amount customizable for different pincodes? =
Yes, you can set different COD charges for each pincode separately.
